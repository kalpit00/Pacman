# Pacman
An exciting game-state based environment involving foundational Artificial Intelligence concepts.

Download the .py files, and run the command : 

python pacman.py

using different flags which test for different mazes (test, tiny, medium, large), different agents(GoWestAgent, GoSouthAgent, etc) and different search algorithms like BFS, DFS, Greedy, A*, Minimax, Alpha-Beta, expectiMax, etc

For example, try 

python pacman.py -l mediumMaze -p SearchAgent -a fn=bfs

to test BFS on a medium sized mazed using the SearchAgent
